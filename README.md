# MC status bot
## RU
Данная программа позволяет сделать telegram бота для проверки статуса сервера и тех людей кто на нём играет, также в данной программе есть функционал сбора статистики в excel таблицу что может быть полезно администраторам серверов в minecraft что бы отслеживать пиковые точки и спады онлайна



# EN
This program allows you to create a Telegram bot to check the status of the server and the people playing on it. This program also has the functionality to collect statistics in an Excel spreadsheet, which can be useful for Minecraft server administrators to track peaks and dips in online activity.


### Installation
1. Install python from official web site
2. `pip install -r requirements.txt` in terminal
2. `python server_status_bot.py` in terminal
